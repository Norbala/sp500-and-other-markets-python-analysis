# S&P 500 (and Other Indices) Analysis

Python scripts for analyzing S&P 500 companies.
For now focusing on SP500, and evaluating them based on financial parameters like cash flows, balance sheet, and current price.
Effectively screen the companies that seem undervalued.

## Features
- Download price data with yfinance
- Create Excel file with data like free cashflow, growth ratios, price movements, etc.
- Analyze sector performance

## Installation
pip install -r requirements.txt

## Usage
python analyze_sp500.py
